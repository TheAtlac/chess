# Игра шахматы для локального использования 2 игроками

## Используемый стек технологий
- Python 2.7
- PyQt 5.15
- SQLite

Для постоянного использования советую пользоваться только main.exe из папки app - 
он обеспечивает всю работу программы.
Его обязательно хранить в одной папке с games, chess_icons и chess_db.sqlite.
Если вам интересен исходный код программ, то его можно найти в папке code.
При запуске exe-файла вам будет предложен выбор сторон, а затем доступен ввод имён участников.
Далее все функции должны быть интуитивно понятны.
